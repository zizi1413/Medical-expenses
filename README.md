# Medical Insurance Charges Analysis (SQL + Python)

## Overview
This project analyzes medical insurance cost data to identify the main factors that influence insurance charges.  
The analysis is performed using **SQL** and **Python (Pandas, Matplotlib, Seaborn)**.

The goal is to extract business insights that could support decision-making in the insurance industry.
---

## Dataset
The dataset contains 1,338 records with the following features:

- Age  
- Sex  
- BMI  
- Children  
- Smoker  
- Region  
- Charges
- 
---

## Business Questions
Key questions explored in this project include:

- How does smoking status affect insurance charges?
- What is the average insurance charge across regions?
- How does BMI relate to charges?
- Do older individuals tend to pay more?
- How does the number of children influence insurance costs?

---

## Tools & Technologies
- PostgreSQL (SQL queries)
- Python (Pandas, NumPy)
- Matplotlib & Seaborn (visualization)
- Jupyter Notebook

---

## Key Insights
- Smokers pay significantly higher charges than non-smokers.
- Age and BMI show positive correlation with charges.
- Southeast region has the highest average insurance cost.
- Obesity category tends to increase charges compared to normal BMI range.
